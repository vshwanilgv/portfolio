# Image Assets

## Directory Structure

- `/hero` - Landing page imagery
- `/projects` - High-resolution project screenshots
- `/tech` - Technology stack logos

## Naming Conventions

- Use lowercase with hyphens: `project-name.jpg`
- Prefer WebP or AVIF formats for better compression
- Recommended sizes:
  - Hero images: 1920x1080px
  - Project cards: 800x600px
  - Tech logos: 200x200px (transparent PNG)
