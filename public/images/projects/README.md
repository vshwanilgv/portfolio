# Portfolio Images

Add your project images here with these filenames:

- `vla-framework.jpg` - Hierarchical VLA Framework project
- `datacanvas.jpg` - DataCanvas IoT platform  
- `gemma-trainer.jpg` - Gemma 3 Trainer project

## Recommended Specifications:
- **Format:** JPG, PNG, or WebP
- **Dimensions:** 800x600px or 1200x900px
- **File size:** < 500KB (Next.js will optimize them)

## Alternative: Use Placeholder Services

Until you have real images, you can use placeholder image services by updating [config/content.ts](../../../config/content.ts):

```typescript
image: 'https://placehold.co/800x600/1e293b/3b82f6?text=Project+Name'
```

Or for better quality:
```typescript
image: 'https://images.unsplash.com/photo-<photo-id>?w=800&h=600'
```
